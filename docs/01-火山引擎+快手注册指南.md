## 第一步：注册火山引擎（即梦 + Seedance 接入）

### 1.1 注册账号

1. 打开 https://console.volcengine.com
2. 点击右上角"注册"
3. 用手机号注册（你的）
4. 选**个人**认证（企业认证要营业执照，咱们先用个人）
5. 实名认证：上传身份证正反面 + 人脸识别

**预计时间：5-10 分钟**

### 1.2 开通视频生成模型

1. 登录后，进入"机器学习平台" → "火山方舟"
   - 路径：https://console.volcengine.com/ark
2. 在"模型广场"搜索：
   - **即梦视频生成** / **jimeng-video**
   - **Seedance 2.0** / **seedance-2-0**
3. 点"开通服务"或"立即体验"
4. 同意服务协议

### 1.3 创建 API Key

1. 进入"访问控制" → "API 访问密钥"
   - 路径：https://console.volcengine.com/iam/keymanage
2. 点击"创建密钥"
3. 命名：天禧派视频助手
4. **保存 Access Key ID 和 Secret Access Key**（只显示一次！）

⚠️ **重要**：把这两个值复制到安全的备忘录里，发给我。我会立刻用它们调通第一个视频。

### 1.4 充值

1. 进入"费用中心" → "充值"
   - 路径：https://console.volcengine.com/finance
2. 充 **200 元**（前期测试够用，扣完可再充）
3. 支付方式：支付宝 / 微信 / 对公转账

**预计时间：5-10 分钟**

## 第二步：注册快手开放平台（可灵 AI 接入）

### 2.1 注册账号

1. 打开 https://open.kuaishou.com
2. 点击"立即入驻"或"开发者中心"
3. 用快手账号登录（没有就注册一个）
4. 选**个人开发者**（企业开发者要营业执照）

### 2.2 创建应用

1. 进入"控制台" → "应用管理"
2. 点击"创建应用"
3. 填写：
   - 应用名称：天禧派视频助手
   - 应用类型：自用 / 工具
   - 简介：家居行业 AI 视频生成
4. 提交审核（通常 1-24 小时）

### 2.3 开通可灵 API

1. 找到"可灵 AI" → "API 服务"
2. 开通可灵视频生成 API
3. 创建 API Key
4. **保存 API Key**

### 2.4 充值

1. 进入"费用中心"或"钱包"
2. 充 **200 元**
3. 支付方式：支付宝 / 微信

**预计时间：15-30 分钟（含审核等待）**
