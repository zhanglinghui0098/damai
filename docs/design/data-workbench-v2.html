设计思路

数据工作台 - 核心定位

不是"选题工具"或"数据看板"——是 AI 的训练场。用户每天用 → 投喂数据（行业 / 城市 / 账号 / 爆款 / 投流）→ 平台输出更准的选题和方向预测 → 用户用画布生成内容 → 平台数据回流 → AI 更聪明。正循环。

① 不绑死画布

用户自拍的视频 / 第三方剪辑的视频也能录入, 手动回填或导数据。哪怕不用画布, 数据工具也要是个合格的新媒体助手。

② 项目-子项目两层结构

大项目 = 一个新媒体账号的一次运营目标（如"顾家家居 618 大促"）；子项目 = 账号下每条视频。数据按这条线汇总, AI 按这条线学习。

③ 优化目标: ROI, 不是点赞

投流用户要的是表单成本 / ROI, 不是完播率。100 万播放 0 转化是垃圾, 5000 播放 50 表单是金矿。AI 的目标函数是这个。
